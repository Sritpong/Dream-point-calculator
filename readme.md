# Dream Point Calculator GUI

โปรแกรม Python แบบ GUI สำหรับคำนวณว่า  
ถ้ามี **Dream Point** จำนวนหนึ่ง ควรซื้อแพ็กไหนบ้างเพื่อให้ **Point เหลือน้อยที่สุด**

ตัวโปรแกรมจะคำนวณโดยอัตโนมัติว่า:
- ใช้ Point ได้มากที่สุดโดยไม่เกินที่มี
- เหลือ Point น้อยที่สุด
- ถ้าเหลือเท่ากัน จะเลือกแบบที่ใช้จำนวนชิ้นน้อยกว่า

---

## Features

- GUI ใช้งานง่ายด้วย `tkinter`
- คำนวณชุดการซื้อที่เหมาะสมที่สุด
- รองรับการกรอก Dream Point ได้โดยตรง
- มีปุ่มตัวอย่างสำหรับทดสอบค่า
- สามารถ build เป็น `.exe` ได้ด้วย `PyInstaller`

## ตัวอย่างการทำงาน

ถ้ามี Dream Point เช่น `10,580`  
โปรแกรมจะคำนวณว่า ควรซื้อแพ็กใดบ้างเพื่อให้เหลือ Point น้อยที่สุด

---

## Requirements

- Python 3.10 ขึ้นไป
- ใช้ไลบรารีมาตรฐานของ Python เป็นหลัก
- GUI ใช้ `tkinter`

> โดยปกติ `tkinter` จะมากับ Python อยู่แล้ว

---

## วิธีรันโปรแกรม

1. ติดตั้ง Python ให้เรียบร้อย
2. ดาวน์โหลดไฟล์โปรเจกต์
3. เปิด CMD หรือ PowerShell ในโฟลเดอร์โปรเจกต์
4. รันคำสั่ง:

```bash
python dream_point_gui.py
